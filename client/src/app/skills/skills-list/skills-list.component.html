Skills I've Learned

Throughout the development of SkillCheck, I’ve honed various skills in both **.NET** and **Angular**. Below are the key skills I’ve developed while building this platform.

Walking Skeleton for Angular and .NET

I learned how to create the basic structure of the project by integrating Angular with .NET, allowing both frameworks to communicate.

Entity Framework

I integrated **Entity Framework** to work with databases, learning to map and manage data in the system seamlessly.

SQLite Database Provider

I implemented **SQLite** as the database provider, learning to configure it with .NET and handle local data storage efficiently.

Context Constructors

I explored how to create and configure **context constructors** to manage database connections and entities efficiently in **Entity Framework**.

Migration

I gained experience with **migration** in Entity Framework, allowing me to modify the database schema without losing data.

API Testing with Postman

I tested APIs using **Postman**, learning how to make requests and verify the responses for effective debugging and validation.

HTTP Requests in Angular

I learned how to make **HTTP requests** from the client-side in Angular, enabling communication between the front-end and back-end systems.

Bootstrap

I implemented **Bootstrap** to create responsive, visually appealing designs, and ensure a smooth user experience across devices.

Authentication with JWT

I integrated **JWT authentication** to manage user sessions securely, ensuring a seamless and safe login experience.

Data Transfer Object (DTO)

I used **DTOs** to transfer data between layers of the application, ensuring efficient and secure data communication between the client and server.
